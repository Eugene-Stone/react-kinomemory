# 🎬 KinoMemory

Веб-приложение для управления фильмами и избранным. SPA на React с авторизацией, системой рекомендаций и функциями администратора.

## 📋 Описание проекта

KinoMemory — это интерактивное приложение для просмотра каталога фильмов с возможностью:
- ✅ Аутентификации пользователей
- ✅ Добавления фильмов в избранное
- ✅ Фильтрации по жанру и рейтингу
- ✅ Сортировки по различным параметрам
- ✅ Поиска фильмов в реальном времени
- ✅ Просмотра информации о фильме (описание, рейтинг, трейлер)
- ✅ Управления профилем пользователя
- ✅ Администрирования каталога фильмов (для админов)

## 🛠 Технологический стек

**Frontend:**
- React 19
- TypeScript
- React Router v7 для маршрутизации
- Context API для управления состоянием
- SCSS для стилизации
- Swiper для карусели
- React Hook Form для форм

**Build & Server:**
- Vite как сборщик
- JSON Server для REST API
- Concurrently для параллельного запуска сервера и приложения

## 🎯 Основные фичи

### 1. **Авторизация**
- Вход в систему через форму
- Сохранение сессии в localStorage
- Выход из системы с очисткой данных

### 2. **Каталог фильмов**
- Динамическая загрузка фильмов с сервера
- Отображение скелетонов во время загрузки
- Обработка ошибок при загрузке

### 3. **Фильтрация и сортировка**
- Фильтр по жанру (динамический список из каталога)
- Фильтр по рейтингу (≥ 9)
- Фильтр "Только избранное"
- Сортировка: по алфавиту, по рейтингу (возрастание/убывание)

### 4. **Поиск**
- Поиск по названию фильма в реальном времени
- Debounce для оптимизации производительности

### 5. **Избранное**
- Добавление/удаление фильмов из избранного
- Проверка на дубликаты перед добавлением
- Синхронизация с сервером через POST/DELETE запросы
- Отдельный фильтр для просмотра только избранного

### 6. **Профиль пользователя**
- Просмотр информации о пользователе
- Для обычных пользователей: рекомендация случайного фильма
- Для администраторов: возможность добавления новых фильмов
- Управление своими добавленными фильмами

### 7. **Модальные окна**
- Глобальный контекст для модальных окон
- Рендер через React Portal
- Закрытие по overlay или кнопке

### 8. **Тема**
- Поддержка светлой и темной темы
- Сохранение выбранной темы в localStorage

## 🔌 API Endpoints

Приложение использует JSON Server, доступные endpoints:

```
GET  /movies           # Получить все фильмы
GET  /movies/:id       # Получить фильм по ID
POST /movies           # Создать новый фильм
DELETE /movies/:id     # Удалить фильм

GET  /favorites        # Получить избранное
POST /favorites        # Добавить в избранное
DELETE /favorites/:id  # Удалить из избранного

GET  /users            # Получить пользователей
GET  /users?login=X&password=Y # Авторизация
```

## 📝 Практики, используемые в проекте
- Типизация с TypeScript
- Разделение логики в кастомные хуки
- Использование Context API для избежания prop drilling
- Управление побочными эффектами через useEffect
- Использование useMemo для оптимизации фильтрации
- Обработка асинхронности
- Скелетоны для улучшения UX при загрузке
- Сохранение состояния в localStorage
- ESLint для проверки кода
